CSS

Existem três formas de aplicar CSS num documento: inline, internamente e externamente

As três formas podem ser usadas e misturadas livremente, e funcionam numa forma de "hierarquia" onde o inline sobrepõe o interno que por sua vez sobrepõe o externo

CSS inline

Para fazer o uso da CSS inline você basicamente chama o parametro style dentro do elemento que você que aplicar e então coloca os parametros da CSS na própria linha do elemento (exemplo no código)

Apenas aplica o estilo ao elemento em que o parametro é chamado

CSS interno/local

O CSS interno é feito na tag "meta" do código, nele você escolhe "divisores" e seleciona estilos para eles

Esse tipo de aplicação funciona para fazer um estilo especifico para uma página especifica, porém não serve para aplicar a toda a documentação do site

CSS externo

O CSS externo é feito num arquivo por fora com a extensão ".css", para aplicar o estilo do arquivo você deve adicionar uma tag "link" à head do documento HTMl e então especificar o nome do arquivo CSS que deve ser utilizado

Essa aplicação é a mais recomendada por ser de mais fácil manutenção e aplicação, podendo ser adicionada fácilmente em várias páginas diferentes

É possível colocar vários links para arquivos CSS na mesma página e elas irão somar umas às outras
