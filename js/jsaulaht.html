Operador térnario

Uma forma de fazer um teste e executar um comando a partir do resultado

        
    nome == 'carlos'?console.log('carlos'):console.log('não carlos');

É basicamente uma linha em três partes: um teste, uma execução pra caso o teste retorne true e outra caso o teste retorne false

Não sei por que usar isso ao invés de condições mas tá aí, talvez tenha melhor performace ou algo assim, vale pesquisar

Forma prática de escrever string

Existe no Javascript uma forma de criar aquelas linhas que mostram variaveis e strings de forma mais fácil, que é a template string, por exemplo:

Linha de string normal:


    var nome = italo
    var idade = 18
    var curso = cc
    console.log('olá! me chamo ' + nome + ' tenho ' + idade + ' anos e estou cursando ' + curso)

o output desse código seria olá! me chamo italo tenho 18 anos e estou cursando cc

Template string:


    var nome = italo
    var idade = 18
    var curso = cc
    console.log(`olá! me chamo ${nome} tenho ${idade} e estou cursando ${curso}`)

este código teria o mesmo output: olá! me chamo italo tenho 18 anos e estou cursando cc

Funções

Existem várias formas de formatar funções no javascript, é possivel criar funções sem nome, com nome, utilizando flechas, tem exemplos na pasta de electron antes dessa de exercicios. Pelo que eu vi, não existe diferença entre essas formatações de função além da visualização mesmo

Recursividade

É quando chamamos uma função dentro dela mesma, um bom exemplo é essa função de fatorial:


    function fatorial(n){
        if(n == 1){
            return 1;
        }else{
            return n * (n-1);
        }
    }

Algumas Funções

É possivel usar as funções "Number(n)" para converter uma variável para um número,independente de ser real ou inteiro, e "String(n)" para converter para uma string

Switch case

O switch serve para situações em que você quer executar uma ação para vários possiveis valores de uma expressão

O if sempre pode substituir o switch mas creio que o switch tenha melhor performace se for um caso de multiplos valores especificos e tal

Estrutura:

        
    switch(x){
        case 0:
            console.log('true')
            break
        case 1:
            console.log('false')
            break
        default:
            console.log('error')
            break    
    }

Esse código basicamente vê o valor de x e se for 0 ou 1 ele retorna verdadeiro ou falso

Se o break não for colocado no código ocorre uma espécie de "efeito cascata" no qual o código vai continuar lendo os cases até encontrar um break, isso pode ser muito util se você quiser que uma sequencia de cases sejam lidos em sequencia por algum motivo

Objetos

Objetos são váriaveis que podem guardar diversas informações e indexar elas da forma que você quiser, arrays por exemplo são objetos porem com indexação e funções fixas

Diferente dos arrays, os objetos são definidos com chaves

exemplo de objeto:

var pessoa = {nome: `carlos`,altura: 1.60, peso: 70, andar(){}}

andar() nesse caso é uma função e pode aceitar parametros para aceitar uma chamada depois:


    var pessoa = {nome: `carlos`,
    altura: 1.60, 
    peso: 70, 
    crescer(crescimento = 0){
        altura+=crescimento
    }}
