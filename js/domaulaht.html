Selecionando elementos do documento

Existem algumas formas de selecionar um elemento do código:

getElementsByTagName()
Essa função retorna todos os elementos cuja tag coincide com o nome fornecido nos parenteses, se você quiser selecionar um elemento especifico entre as tags é possivel fornecer um numero, utilizando colchetes, que escolherá o elemento baseado na ordem que ele aparece no código, getElementsByTagName('p')[1] por exemplo retornaria a linha "olha o teste..."
getElementById()
Essa função retorna um elemento com um id especifico, esse id deve ser adicionado à tag em especifico que você quer que seja selecionada no html, getElementById('divisao') por exemplo retornaria a divisória que envelopa a linha "mais teste"
getElementsByName()
Basicamente a mesma coisa do Id porém utilizando o parametro "name" que, diferentemente do Id, não precisa ser único para todo o documento e pode ser usado em várias tags diferentes
getElementsByClassName()
querySelector
Basicamente uma mistura do id com o tagname, ele pede tanto a tag quanto o id ou a classe, o que for, querySelector('div#divisao') seleciona aquela div. Caso o parametro da div seja "class" e não "id" o código seria querySelector('div.divisao').

Todas essas funções são partes do elemento "document"

Eventos
