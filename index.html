Links

Links externos

utilizando a tag "a" é possivel colocar um link para qualquer site, até esse mesmo

essa tag aceita parametros como "target" que permite que o link seja aberto em uma nova guia ou "rel" que especifica a relação desse documento com o do link(semântico)

Links internos

É também possivel usar os links para mandar para uma outra página da própria documentação interna: Formatações do HTML

Também aceita parametros, nesse exemplo o rel indíca que a página do link seria a próxima na sequencia de visualização

Para acessar um documento de uma pasta parente é necessário utilizar "../" que significa basicamente pasta anterior


É sempre importante especificar o rel do link, existem muitas tags, no vscode elas podem ser vistas colocando o mouse em cima do rel e clicando em link types
