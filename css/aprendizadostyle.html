Coisas gerais de CSS

É possivel usar "*" como tag no arquivo css para inserir propriedades globais

É possivel definir configurações especificamente para elementos que são filhos de um elementoe specifico, a linha div > p{} por exemplo, configura apenas paragrafos dentro de divs

Existe um shorthand para definir paddings, margens e etc, apenas use a função normalmente e escreva os valores na seguinte ordem: top, right, bottom e left

Cores em CSS

Parte importante do design de um site é sua paleta de cores, para isso é necessário entender a harmonia de cores: Pdf com tabela sobre significado das cores

O site Adobe Color é uma ótima ferramenta para gerar paletas de cores

Também é possivel, no adobe color, inserir imagens para pegar a paleta de cores delas e fazer gradientes com a paleta que você quiser

Além disso existe a área de explorar, onde é possivel procurar paletas de cores muito usadas e copiar elas em css, ou só pegar inspiração

No site palleton.com você pode colocar uma paleta de cores e vizualizar exemplos, incluindo um template de site, utilizando sua paleta

Váriaveis CSS

É possivel criar variáveis em CSS, elas podem ser muito uteis para guardar uma paleta de cores ou coisas do tipo, para criar uma variável você deve usar a pseudo-classe ":root" na CSS e então utilizar "--nomedavariavel: valor;""

Tipografia

Ao colocar a fonte no "font-family" do CSS pode ser necessário colocar aspas para fontes que tem duas palavras

Serifa

A serifa serve basicamente pra ajudar na leitura mas em meios digitais polui muito a tela

Fontes sem serifa devem ser priorizados ao utilizar em textos longos em sites, deixando as fontes com serifa para documentos destinados a serem impressos em papel

Tamanho das fontes

Existem diversas medidas que podem ser utilizadas para determinar o tamanho de uma fonte, elas podem ser medidas absolutas, como o pixel(px) ou centimetros(cm) ou relativas, como a relativa ao tamanho da letra M da fonte(em). É recomendado utilizar medidas relativas para fontes e se não, utilize pixels

Geralmente 16px = 1em

Funções CSS para fontes

font-weigth
define se a espessura da fonte, light, bold e etc
font-style
define o estilo da fonte, itálico e tal
text-decoration
aplica decorações como por exemplo sublinhado, usando underline

É possivel ainda utilizar apenas a função font, que define o estilo, espessura, tamanho e familia respectivamente. Exemplo: 'Work Sans', sans-serif; A ordem não pode ser mudada

Importando fontes

É possivel encontrar e importar fontes a partir do Google Fonts

É possivel também importar fontes baixadas utilizando o seletor @font-face

A extensão "fontsninja" permite descobrir qual a familia e outras propriedades de qualquer fonte de qualquer site da web, a extensão pode ser encontrada no chrome web store

Classes e ID

Classes podem ser usadas mais de uma vez, id's precisam ser únicos

É possivel adicionar duas classes para um unico elemento, ou até adicionar um id e uma classe no mesmo elemento, porém o id sempre terá prioridade

Um uso que achei interessante para as classes é criar uma classe que destaca uma parte do texto e usar a tag "span" com essa classe para destacar

Pseudo-Classes

São espécies de funções que podem ser ativadas em alguns eventos em especificos e assim mudar a CSS do site, são escritas usando ":"(dois pontos) depois do nome da tag

Pseudo-Elementos

São também meio que funções porém ao invés de mudar a CSS do documento ele muda o conteúdo do próprio elemento em HTML, eles são usados com "::"(dois dois pontos)

Modelo de caixa

É o modelo utilizado pelo CSS para posicionar os elementos na tela

Existem elementos box-level, que ocupam toda a largura do body, como por exemplo divs ou articles, já os elementos inline-level apenas ocupam o espaço necessário para mostrar seu conteúdo, como por exemplo spans ou strongs

Layout

É importante definir o layout de um site antes de começar a produzi-lo de fato, para isso existem ferramentas como o Mockflow, que é pago mas tem versão gratuita

Responsividade

A largura máxima ideal de um bloco que comporta texto em HTML é cerca de 1200 pixels, passando disso deve ser criado um bloco que se prenda nesse tamanho
