imagem dinamica

imagem que aumenta e diminui de tamanho dependendo da largura do navegador, bom para evitar que o usuário precise usar rolagem horizontal para ver a imagem inteira

para fazer isso utilizamos a tag "picture" que é como uma lista que aceita parametros (parecidos com ifs) e mostra as imagens dependendo desses parametros

audios

a melhor forma de implementar audio em sites é utilizando javascript

para os audios podemos utilizar a tag "audio", podendo ser utilizada como uma lista de sources ou apenas um, é melhor usar como lista e fornecer varios tipos de arquivo de audio pois nem todos os navegadores tem compatibilidade com todos os tipos de arquivos

pelas minhas pesquisas por fora, uma forma mais segura de fazer o autoplay e tudo mais funcionar é usando javascript

videos

videos hospedados localmente

um grande problema de hospedar videos localmente é que o custo da hospedagem de videos é muito alta e vai ter que sair do seu bolso

para adicionar videos ao site utilizamos a tag "video" de forma bem parecida com a de audio, colocando sources e etc

para videos também é recomendavel usar vários formatos de vídeos, apesar que eu não tou fazendo isso aqui, mas faça

incorporando videos

as duas melhores opções para incorporação de videos são o Vimeo, cujas vantagens são a melhor qualidade e maior personalização, e o Youtube, cujas vantagens são melhor codec e ser mais conhecido

para incorporar videos é possivel apenas entrar nesses sites de video, fazer upload do video que voce quer e compartilhar ele, vai aparecer a opção de incorporar e daí é só copiar pro código do seu site
