É possível utilizar os atributos rowspan e colspan na tag HTML para determinar quantas colunas e linhas cada célula irá ocupar

O atributo scope precisa ser dado com group caso o colspan ou o rowspan de uma célula seja aumentado e ele entitule mais de uma célula de dado

Para selecionar colunas para mudanças de estilo por exemplo é necessário utilizar a tag colgroup para determinar quantas e quais são as colunas da tabela, é possivel usar o atributo span para determinar um grupo de colunas, essas colunas podem receber classes e etc
